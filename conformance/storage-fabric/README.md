# Storage Fabric Conformance

This subtree carries the executable conformance surfaces for the storage-fabric standards.

It is intentionally downstream of the standards authority.

Upstream normative sources:
- `SocioProphet/prophet-platform-standards` for platform doctrine
- `SocioProphet/socioprophet-standards-storage` for benchmark methodology and report schemas

This subtree provides:
- minimal reference implementations for history and ranking interpretation
- validation runners for generated benchmark and ranking reports
- generated example fixtures used to verify conformance behavior

This subtree does **not** redefine the platform standard. It proves and exercises it.
